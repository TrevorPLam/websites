# Web App - Your Dedicated Marketer

This is the main Next.js application for the marketing website.

## Framework
- **Next.js 14+** with App Router
- **React Server Components** by default
- **TypeScript** for type safety

## Structure
- `app/` - Next.js App Router pages and routes
- `components/` - App-specific components (not in design system)
- `features/` - Feature modules
- `lib/` - App-specific utilities

## Key Files
- `app/layout.tsx` - Root layout
- `app/page.tsx` - Homepage
- `app/globals.css` - Global styles

## Development
- Run: `npm run dev` or `pnpm dev`
- Build: `npm run build`
- Uses Server Components by default, Client Components when needed
