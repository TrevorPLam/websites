# Anthropic Skills - MCP Builder + Playwright + Document Skills

## Overview
This directory contains skills for working with Anthropic's tools and services, including MCP building, Playwright testing, and document generation capabilities.

## Available Skills

### MCP Builder
- **mcp-build**: Build and deploy Model Context Protocol servers
- **mcp-test**: Test MCP server functionality
- **mcp-deploy**: Deploy MCP servers to production

### Playwright Testing
- **playwright-test**: Run Playwright test suites
- **playwright-generate**: Generate Playwright tests from specifications
- **playwright-analyze**: Analyze test results and performance

### Document Skills
- **doc-generate**: Generate documents from templates
- **doc-convert**: Convert between document formats
- **doc-analyze**: Analyze document content and structure

## Usage
These skills can be invoked by AI agents to automate development workflows, testing, and documentation tasks.

## Configuration
Ensure proper API keys and permissions are configured for Anthropic services.

## Dependencies
- Anthropic API access
- Playwright testing framework
- Document processing libraries
