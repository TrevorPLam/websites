# UI Design System Package

This package contains the shared design system components used across the application.

## Structure
- `src/components/` - React components (Button, Card, Input, etc.)
- `src/styles/` - Global styles and CSS
- `src/tokens/` - Design tokens (colors, spacing, typography)

## Usage
Import components from this package:
```typescript
import { Button } from '@repo/ui'
```

## Guidelines
- All components should be reusable and framework-agnostic where possible
- Use TypeScript for type safety
- Follow the design system tokens for consistency
