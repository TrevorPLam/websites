# Contributors

**Last Updated:** 2026-02-18  
**Status:** Active Recognition

---

We're grateful to all contributors who help improve the marketing-websites platform documentation. This page recognizes those who have made significant contributions.

## Documentation Contributors

### 2026

**Documentation Excellence Initiative**
- Comprehensive documentation improvements
- New tutorials and guides
- Enhanced visual documentation
- Improved discoverability

## How to Contribute

See [CONTRIBUTING.md](../CONTRIBUTING.md) for guidelines on contributing to documentation.

## Recognition Criteria

Contributors are recognized for:
- Significant documentation improvements
- New documentation sections
- Major updates or rewrites
- Accessibility improvements
- Translation contributions
- Long-term maintenance

## Adding Contributors

To add a contributor:
1. Get their consent
2. Add their name/GitHub username
3. Describe their contribution
4. Update this file in your PR

**Thank you to all contributors!** Your efforts make this project better for everyone.
