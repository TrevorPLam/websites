# 23.1 Philosophy

**What it is:** For a local service business, SEO is the revenue engine â€” not a nice-to-have. The platform must generate correct `generateMetadata()`, `sitemap.ts`, `robots.ts`, JSON-LD structured data, and OpenGraph images automatically from the tenant's `site.config`, requiring zero manual SEO work from the client. [jsdevspace.substack](https://jsdevspace.substack.com/p/how-to-configure-seo-in-nextjs-16)

**Why automatic matters:** A plumber doesn't know what `LocalBusiness` schema is. The platform reads their config and emits perfect structured data on every page. Google parses the `LocalBusiness` schema to populate the Knowledge Panel, `HoursSpecification` to show business hours in Maps, and `Service` markup to power rich results. Missing any of these costs clients meaningful organic traffic. [dev](https://dev.to/ared/how-to-add-json-ld-structured-data-in-nextjs-for-seo-52i0)

**The three layers:**

| Layer              | File                              | What It Controls                                         |
| ------------------ | --------------------------------- | -------------------------------------------------------- |
| Technical metadata | `layout.tsx` `generateMetadata()` | Title, description, canonical, robots, OG, Twitter cards |
| Crawl directives   | `robots.ts`                       | Which pages Googlebot indexes; block portal routes       |
| Site structure     | `sitemap.ts`                      | All indexable URLs with `lastModified` and `priority`    |
