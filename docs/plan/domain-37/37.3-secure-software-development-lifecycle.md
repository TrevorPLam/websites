- DOMAIN-37-4-1 select SAST tooling for repository workflows (CodeQL + Semgrep).

## QA checkpoints for this parent task

1. `pnpm validate-docs` passes after adding threat modeling and policy documents.
2. `node scripts/verify-staged-secrets.js` passes on clean staged changes and fails on obvious secret patterns.
3. Security requirement docs cross-reference standards mapping, secure coding guidance, and threat model artifacts.
4. `.github/workflows/security-sast.yml` runs CodeQL and Semgrep scans on push, PR, and schedule.
5. `.husky/pre-commit` executes security checks before lint-staged formatting.
