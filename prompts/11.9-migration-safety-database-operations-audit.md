# 11.9 Migration Safety & Database Operations Audit

```text
MODE: PLAN

Audit database migration safety and operational integrity.

Check:
- Destructive migration detection and prevention
- Backward compatibility guarantees
- Tenant-safe schema evolution patterns
- Index efficiency and locking risks at scale
- Migration rollback capabilities
- Data integrity during schema changes
- Migration testing in staging environments
- Migration execution time and performance impact
- Schema change communication and coordination
- Migration failure recovery procedures

Deliver:
- Migration safety score
- Production deployment risk assessment
- Rollback capability evaluation
- Performance impact analysis
- Operational readiness checklist
```
